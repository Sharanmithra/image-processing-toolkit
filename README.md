# 📷 Image Processing Toolkit using OpenCV

## 📌 Project Overview

This project is a **Python-based Image Processing Toolkit** built using **OpenCV**. It allows users to perform basic image processing operations such as resizing, grayscale conversion, edge detection, and blurring.

The toolkit demonstrates how computer vision techniques can be applied to process and analyze images efficiently.

---

## 🎯 Objectives

* Load images from file paths
* Resize images to fixed dimensions
* Convert images to grayscale
* Detect edges using Canny edge detection
* Apply Gaussian blur filters
* Save processed images automatically

---

## ⚙️ Technologies Used

* Python
* OpenCV

## 🚀 How to Run This Project

### Step 1 — Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 2 — Run the Script

```bash
python image_toolkit.py
```

### Step 3 — Provide Image Path

Example:

```text
Enter image path: sample.jpg
```

The system will generate:

* resized.jpg
* grayscale.jpg
* edges.jpg
* blurred.jpg

These files will be saved in the **output** folder.
